# Archived Roadmaps

This directory contains historical roadmap versions for audit trail and reference.

## Versioning Scheme

Files are named: `{DOCUMENT}_v{VERSION}_{DESCRIPTION}_{DATE}.md`

## Archive Contents

| File | Date | Description |
|------|------|-------------|
| `ROADMAP_v1.0_phases-0-10_2025-12-06.md` | 2025-12-06 | Original Phases 0-10 roadmap |
| `ROADMAP_EXTENDED_v1.0_2025-12-06.md` | 2025-12-06 | Extended roadmap (Phases 7-10) |
| `CURRENT_WORK_v1.0_2025-12-06.md` | 2025-12-06 | Session context file |

## Archive Policy

1. **When to archive**: Before major roadmap changes
2. **What to archive**: ROADMAP.md, ROADMAP_EXTENDED.md, CURRENT_WORK.md
3. **Retention**: Indefinite (for audit trail)
